Let's install [oidc-spa](https://github.com/keycloakify/oidc-spa) in your project: &#x20;

{% tabs %}

```bash

```
{% endtab %}

{% tab title="yarn" %}
```bash
yarn add oidc-spa
```
{% endtab %}

```bash
pnpm add oidc-spa
```
{% endtab %}

{% endtab %}
{% endtabs %}
